# Knigavuhe Enhanced

Userscript с небольшими улучшениями удобства для `knigavuhe.org`.

## Возможности

- Добавляет компактный инфоблок в плеер сайта.
- Показывает название активной главы.
- Показывает общее прослушанное время: сумма предыдущих глав плюс прогресс текущей.
- Читает полную длительность книги со страницы, когда она доступна.
- Сохраняет визуальный стиль оригинального плеера легкими CSS-доработками.

## Установка

1. Установите менеджер userscript-скриптов, например Tampermonkey или Violentmonkey.
2. Откройте `knigavuhe-enhanced.user.js`.
3. Установите или обновите скрипт через интерфейс менеджера.

## Разработка

Редактируйте `knigavuhe-enhanced.user.js` напрямую. У скрипта намеренно нет сборки и зависимостей.

Поля метаданных, которые стоит обновлять при необходимости:

- `@version` при публикации изменений.
- `@grant`, если понадобятся browser/userscript API.
